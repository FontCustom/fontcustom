FontCustom
==========

__Automated glyph font generator. Stop building glyph fonts by hand and FontCustom-ize.__


Requirements
------------
FontForge
  On OS X: ``brew install fontforge``


Usage
-----
    fontforge -script generate.py path/to/svgs
